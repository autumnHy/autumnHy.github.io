模板配置&使用规范

一. 创建模板

  1. 手动创建 和

  1. 在 settings.py 添加 模板地址

  2. 创建 index.html 的路由

    注: 一个项目肯定会有多个应用(app) 列如 登陆的 业务的 等 ,路由条目会很多,所以主目录下的urls.py 不易存放全部的路由,应该将 模板放在对应的应用文件下面方便管理.

    2.1 在 __应用文件(app)__ 下创建所属 urls.py 文件

    2.2 再在 __主目录的urls.py__ 里创建跳转到 应用文件下的urls.py 路由条目

    2.3 最后在 __应用文件下的 views.py __里添加跳往 index.html 的方法

    设置完成过后在开启服务器在浏览器中输入 ip:port/app/index.html 就可以跳转到index.html.

  1. 创建文件夹 static 

    1.1 再在 **static文件夹里** 创建 css / js / image 三个子文件夹

  2. 设置静态资源文件夹 (static) 的路径

    settings.py 文件里添加路径
