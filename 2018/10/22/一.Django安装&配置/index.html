Django安装与配置

一. 安装虚拟环境

在安装Django前先创建一个纯净的Python虚拟环境

  1. 安装 virtualenv

    virtualenv 是用来安装虚拟环境的

    pip3 install virtualenv

  2. 在 新文件夹中 里执行命令创建一个干净的虚拟环境

    创建一个新文件夹专门用来存放每个工程所需的环境

    virtualenv --no-site-packages -p <Python目录> <新文件夹名>

    注: 忘记Python安装目录可以在环境变量里查看

  3. 启动环境变量

    Linux：　source bin/activate 退出source deactivate

    Windows: 　cd /<新文件夹>/Scripts 执行activate 退出deactivate

  4. 查看当前环境下的库

    开启虚拟环境后 看下当前虚拟环境下的库

    pip list

二. 安装 Django

  1. 在 虚拟环境Scripts目录 里安装django指定版本

    pip install django==1.11

    注: python3.6 对应 django1.11

  2. 切换到 新的项目目录里 并 创建Django工程

    django-admin startproject < name_project >

    注: 创建完成后会有4个文件夹以及1个manage.py文件

  3. 将当前工程添加到新的虚拟环境下去执行

    3.1 PyCharm –> File –> Open –> < name_project >

    3.2 PyCharm –> Settings –> Project < name_project > –> Project Interpeter

    3.3 Project Interpeter –> Show all –> < 新虚拟环境>/Scripts/python.exe –> OK
