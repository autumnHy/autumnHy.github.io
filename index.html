Django安装与配置

一. 安装虚拟环境

  1. 安装virtualenv

    pip3 install virtualenv

  2. 在 新文件中 里执行命令创建一个干净的虚拟环境

    virtualenv --no-site-packages -p <Python目录> <新文件名>

    忘记Python安装目录可以在环境变量里查看

  3. 启动环境变量

    Linux：　source bin/activate 退出source deactivate

    Windows: 　cd Scripts 执行activate 退出deactivate

  4. 查看当前环境下的库

    pip list

二. 安装 django

  1. 在 虚拟环境Scripts目录 里安装django

    pip install django==1.11

    python3.6 对应 django1.11

  2. 切换到 新的项目目录里 并 创建django项目

    django-admin startproject <项目名称>

    项目目录里会生成4个文件夹:

    __intit__.py: 初始化,配置pymyaql连接的地方

    settings.py: 配置信息位置,database等

    urls.py: 配置URL路由

    wsgi.py : 配置网关
    注: 项目文件里添加新虚拟环境的解释器

三. 设置 django

  1. 在 settings.py 里设置

  1. 安装 pymysql 库

    pip instarll pymysql

  2. 在 __init__.py 里添加 mysql 驱动

  1. 迁移表格

    Terminal > python manage.py migrate

注:会在数据库里导入表格

  1. 创建超级管理员用户

    Terminal >python manage.py createsuperuser

  2. 启动 django

    Terminal > python manage.py runserver <ip:prot>
